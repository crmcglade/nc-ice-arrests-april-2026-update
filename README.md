# North Carolina ICE Arrests Analysis

This repo analyzes North Carolina arrests for the latest batch of [Deportation Data Project](https://ice-arrests.apps.deportationdata.org/) data.

You can find the raw files [here](https://ice-arrests.apps.deportationdata.org/).

To download my processed data that includes all North Carolina arrests, along with additional columns flagging certain events such as the Border Patrol November raid on Charlotte, go to "processed-data" and download the raw file.
