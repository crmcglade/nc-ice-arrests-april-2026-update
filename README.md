# North Carolina ICE Arrests Analysis

This repo analyzes North Carolina arrests for the latest batch of [Deportation Data Project](https://ice-arrests.apps.deportationdata.org/) data.

To access my processed data that includes all North Carolina arrests, along with additional columns flagging certain events such as the Border Patrol November raid on Charlotte, [go here.](https://raw.githubusercontent.com/crmcglade/nc-ice-arrests-april-2026-update/refs/heads/main/processed-data/nc-arrests-april-26-update.csv)
